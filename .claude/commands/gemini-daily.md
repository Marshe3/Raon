# Gemini 일일 분석 결과 활용

최신 Gemini CLI 일일 분석 보고서를 읽고 Claude Code가 처리할 수 있는 액션 아이템을 실행하세요.

## 단계

1. `.ai-workflow/gemini-output/daily-analysis/latest.md` 파일 읽기
2. 분석 결과 요약 제시
3. "Claude Code 액션 아이템" 섹션의 작업들을 우선순위별로 정리
4. 각 항목에 대해:
   - 자동으로 처리 가능한지 판단
   - 가능하면 즉시 처리
   - 불가능하면 이유 설명 및 대안 제시
5. 처리 결과 요약
